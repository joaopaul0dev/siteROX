# Landing Page - ROX BJJ Planalto

Este projeto é uma landing page institucional desenvolvida para a academia ROX BJJ Planalto, com foco em apresentação da marca, atração de novos alunos e conversão por meio de contato via WhatsApp.

A página foi criada para mostrar a identidade da academia, destacar seus diferenciais, apresentar horários, planos e facilitar o agendamento de aulas experimentais.

## Objetivo

Criar uma presença online profissional, moderna e otimizada para conversão, permitindo que o cliente:

- apresente a academia de forma atrativa;
- gere interesse por parte de potenciais alunos;
- incentive o contato direto pelo WhatsApp;
- tenha uma página rápida, responsiva e fácil de manter.

## Sobre o projeto

A landing page foi desenvolvida com uma estrutura simples e eficiente, utilizando tecnologias web clássicas para garantir boa performance, compatibilidade e facilidade de hospedagem.

### Características principais

- layout moderno e responsivo;
- navegação simples e intuitiva;
- destaque para chamada para ação (WhatsApp);
- seções para apresentação da academia, horários, planos e FAQ;
- estrutura otimizada para SEO básico;
- pronta para adaptação ao domínio e hospedagem oficial do cliente.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (vanilla)
- Ícones e fontes externas

## Estrutura do projeto

```text
assets/
  images/
  script/
  style/
index.html

```

## Como visualizar localmente

1. Baixe ou clone o projeto.
2. Abra o arquivo index.html em um navegador.
3. Para uma experiência mais próxima da hospedagem, você pode utilizar um servidor local simples, como o Live Server no VS Code.

## Ajustes importantes antes de publicar

Antes de colocar o site no ar no domínio oficial, é recomendável revisar e atualizar os seguintes itens:

- nome do domínio e URLs internas;
- número de WhatsApp e links de contato;
- redes sociais;
- endereço completo e horários;
- imagens de destaque e identidade visual;
- meta tags, título e descrição para SEO;
- arquivo sitemap.xml e robots.txt.

## Hospedagem e domínio

Este projeto é uma página estática e pode ser hospedado em qualquer servidor que suporte HTML/CSS/JavaScript.

Opções comuns:

- GitHub Pages
- Netlify
- Vercel
- Hostinger / hospedagem tradicional

Ao migrar para o domínio oficial, substitua os links internos e externos para o novo endereço e confirme se todos os arquivos de imagem e SEO estão apontando corretamente.

## Observações para o cliente

Este projeto está pronto para ser adaptado ao domínio e à hospedagem oficial da academia. A estrutura é simples, leve e de fácil manutenção, permitindo futuras alterações sem depender de uma grande infraestrutura.
